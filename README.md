# 練習サイト-01

## サイトタイトル

## 概要
* 練習用のWebサイト
* 「slick.min.js」を使用しトップページにスライドショーを実装
* 薄く背景画像を使用しモダンなデザインを意識
* 透過画像に影を残し立体感を演出
